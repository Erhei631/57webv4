Overview

57Blocks partnered with the Stellar Development Foundation across multiple programs to design and ship a suite of practical, adoption-friendly tools that reduce operational risk, improve developer velocity, and strengthen the end-user DeFi experience on Stellar.

What 57Blocks delivered

57Blocks delivered four initiatives across SCF rounds and SDF’s Kickstart program.

Timelock Smart Contract

Goal

Add timelock protection through a non-intrusive approach to improve operational security and governance.

Key capabilities

  • No-code-intrusion integration approach
  • Queue and delay mechanism for scheduled execution
  • Role-based access control
  • Security-oriented design for governance operations

Outcome

A deployable timelock pattern that helps projects reduce upgrade and admin-action risk while keeping adoption friction low.

View program

AutoAction Developer Tool

Goal

Enable developers and operators to schedule and trigger workflows spanning on-chain and off-chain systems.

Key capabilities

  • Scheduled execution of smart contract actions
  • Event-driven action execution
  • External API integration and composability with other contracts

Outcome

A developer-friendly automation layer that reduces manual operations and enables richer protocol workflows.

View program

Resource Usage Monitoring

Goal

Provide real-time insight into resource usage to help teams test, debug, and prevent overuse-related failures.

Key capabilities

  • Test connectivity directly to localnet
  • Alerts for resource overuse
  • Easy, non-intrusive integration into developer workflows

Outcome

Improved feedback loops for teams shipping smart contracts and applications, with earlier detection of performance and resource risks.

View program

Tokenpad

Goal

Provide an end-user app to track Stellar assets in one place and help users understand their DeFi positions.

Key capabilities

  • Consolidated and accurate asset visualization
  • Categorized and analyzed DeFi assets
  • Unified PnL insights to evaluate performance

Outcome

A user-facing portfolio experience that improves transparency and usability for DeFi participants on Stellar.
